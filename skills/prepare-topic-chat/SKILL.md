# Prepare Topic Chat

Turn a broad topic into a conversation scaffold that is useful for later script writing.

## Output

Return a compact preparation note with these sections in plain prose or short bullets:

- Core question the conversation should answer
- 3 to 5 promising angles
- Missing facts or definitions worth clarifying
- Concrete examples, events, papers, books, or cases to bring up
- A suggested questioning path for the next few turns

## Style

Be sharp, specific, and practical.
Prefer concrete angles over generic brainstorming.
Do not write the final script here.
